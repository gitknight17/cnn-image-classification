# Image Classification using CNNs (Transfer Learning)

This project implements an end-to-end image classification pipeline using Convolutional Neural Networks and transfer learning.

## Key Highlights
- Used a pretrained CNN model for image classification
- Implemented image preprocessing and data loading using Torchvision
- Built training and validation workflows using PyTorch Lightning
- Applied early stopping and model checkpointing
- Evaluated model performance using classification metrics

## Tech Stack
- Python
- PyTorch, Torchvision
- PyTorch Lightning
- Torchmetrics
- Jupyter Notebook

## Notes
This project was completed as part of academic coursework.
